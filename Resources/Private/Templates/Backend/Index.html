Bitte geben Sie oben Ihre Assistant ID ein, um den Chatbot zu aktivieren.

🔑 Wo finde ich die Assistant ID?

  1. Melden Sie sich bei my-chatify.de an.
  2. Gehen Sie zum Dashboard und wählen Sie Ihren Assistenten aus.
  3. Kopieren Sie die Assistant ID (z.B. f737786eeaaf8).
  4. Fügen Sie die ID oben ein und klicken Sie Speichern.

🤖 Chatbot erstellen & trainieren

  1. Registrieren: Erstellen Sie ein kostenloses Konto auf my-chatify.de.
  2. Assistent anlegen: Klicken Sie im Dashboard auf „Neuen Assistenten erstellen“ und vergeben Sie einen Namen.
  3. URLs hinzufügen: Geben Sie die URLs Ihrer Website ein – der Chatbot holt sich automatisch die Inhalte (URL-Fetching).
  4. Dokumente hochladen: Optional können Sie auch PDFs, Word- oder Text-Dateien hochladen.
  5. Training starten: Klicken Sie auf „Training starten“ – die Inhalte werden in Chunks aufgeteilt und als Embeddings gespeichert.
  6. Testen: Nutzen Sie den integrierten Chat im Dashboard, um den Bot zu testen.
  7. Einbinden: Kopieren Sie die Assistant ID und fügen Sie sie hier oben ein – fertig!
ⓘ
Hinweis: Nach dem Speichern wird das Chatbot-Widget automatisch auf allen Seiten Ihrer TYPO3-Website eingebettet. Leeren Sie ggf. den TYPO3-Cache, damit die Änderungen sofort sichtbar werden.
